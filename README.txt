APPLICATION EXAMEN CIVIQUE - FOYER LUZEL - VERSION 2

Contenu :
- index.html : application principale
- questions.js : banque de 80 questions propres et structurées
- app.js / styles.css : logique et design
- manifest + service-worker : installation PWA et hors connexion

Utilisation simple :
1. Décompresser le ZIP.
2. Ouvrir index.html dans Chrome, Edge ou Safari.
3. Pour l'installation tablette comme application, déposer ce dossier sur un hébergement HTTPS.

Fonctions :
- Entraînement par thème
- Examen blanc 40 questions
- Révisions automatiques des erreurs
- Favoris
- Recherche
- Cartes mémoire
- Tableau de bord accompagnateur
- Export CSV et JSON
- Impression du bilan
- Mode sombre, contraste élevé, taille de texte, lecture vocale

Données :
Tout est sauvegardé localement dans le navigateur de la tablette. Aucune donnée n'est envoyée en ligne.

Personnalisation :
Vous pouvez modifier les questions dans questions.js.
